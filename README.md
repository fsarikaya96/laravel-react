# Laravel React Todo App

## Install

**1. Clone Project**

```
  $ git clone https://github.com/fsarikaya96/laravel-react.git
  $ cd laravel-react
```
**2. Install**

```
  $ cp .env.example .env
  $ composer install
  $ php artisan key:generate
  $ php artisan migrate 
  $ php artisan db:seed
```

**3. npm Install**

```
  $ npm install
  $ npm install axios
  $ npm install react-router-dom
  $ npm install sweetalert --save
```

**4. Run Project**

```
  $ npm run dev
  $ php artisan serve
```
